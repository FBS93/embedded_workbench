# Embedded Middleware Framework (EMF) overview

EMF is a lightweight, platform-agnostic middleware framework designed to run on both embedded systems and host environments. Its purpose is to provide:
- Lightweight middleware optimized for embedded environments.
- Deterministic and low-overhead execution.

# Glossary

| Term | Definition |
|------|------------|
| Middleware | Reusable software that solves one focused problem and is completely platform-agnostic. |
