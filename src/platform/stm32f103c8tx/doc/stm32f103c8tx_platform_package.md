# STM32F103C8Tx platform package overview

STM32F103C8Tx platform package provides support for embedded targets based on the STM32F103C8Tx series.

See [stm32f103c8tx_stdio.md](../stdio/doc/stm32f103c8tx_stdio.md) and the other source files in this platform package to understand its structure and how to add additional platform packages.

# Usage example

Can be enabled at build time as follows:

```cmake
set(ECF_TARGET_PLATFORM stm32f103c8tx)
```
