# Embedded Platform Framework (EPF) overview

EPF is a lightweight, platform-dependent framework designed to run on specific platforms. Its purpose is to provide:
- Platform-specific libraries and utilities.
- Non-portable APIs tailored to a given platform.
- Optimized implementations leveraging platform-specific features.
